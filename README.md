Retail Sales Data Analysis 🛒

Objective 🎯:
To analyze the sales performance of a retail store

Dataset 🛢️:
Retail sales dataset (1000 transactions)

Tools 🔨:
- Python
- Pandas
- Matplotlib
- Seaborn

Analysis performed 🔎:
- Sales by product category
- Sales by gender
- Sales by month
- Age distribution
- Top customers

Conclusions 📌: 

- Sales by Product category
